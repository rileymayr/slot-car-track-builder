# Slot Car Track Builder

This tool allows users to design complex, multi-lane slot 
car tracks with accurate geometry, elevation changes, and required pieces tracking.

- Currently supports the following track systems:
  - Aurora


# Application Accessible at:
https://rileymayr.github.io/slot-car-track-builder/


## Features

- **Multi-Lane Support**: Design and manage multiple track lanes simultaneously.
- **Accurate Geometry**: Built-in support for standard Aurora track pieces including straights, curves (3", 6", 9", 12", 15", 18" radii), and ramps.
- **Lane Length Calculation**: Real-time calculation of lane lengths for each track lane, including delta comparisons to help balance your layout.
- **Bill of Materials (BOM)**: Automatically generates a list of required track pieces as you build.
- **Table Drawing Mode**: Define your available space by drawing a custom table shape to ensure your track fits.
- **Elevation Support**: Includes "Ramp Up" and "Ramp Down" pieces to create multi-level layouts.
- **Snap-to-Grid**: Intuitive snapping for angles (45° increments) and positions to make track connection seamless.
- **Interactive Editor**: Add, remove, replace, and undo track pieces with ease.


## How to Use

1. **Draw Table (Optional)**: Use the "Draw Table" mode to outline your physical layout area. Click to place corners and "Done" to finish.
2. **Place Starting Point**: Click anywhere on the canvas or table to place your track's starting marker.
3. **Add Pieces**: Use the piece library on the left to choose from the Aurora track library.
4. **Edit Track**:
   - **Click** a piece to select it.
   - **Replace**: Switch an existing piece with another while maintaining the rest of the track's flow.
   - **Insert**: Add a piece before the selected one.
   - **Remove**: Delete a specific piece from your layout.
5. **Manage Lanes**: Use the "Tracks" panel on the right to add new lanes or switch between editing different lanes.
6. **View Stats**: Check the bottom panel for real-time lane lengths and the complete Bill of Materials.
